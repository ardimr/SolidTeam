# Solid Team Charts
A collection of Solid Team's Helm charts for Project Sprint Dev Batch 4

## Prerequisites
- Kubernetes 1.30+
- Helm3

## Quick Start

### Add Helm Repo
- salty-spitoon
    ```bash
    helm repo add salty-spitoon-repo https://github.com/ardimr/SolidTeam/salty-spitoon
    ```

    ```bash
    helm repo update
    ```
